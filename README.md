### Description :
Application de détection de fake news utilisant un modèle SVM entraîné et l'API Mistral AI. Interface développée avec Streamlit.

### Technologies :
Streamlit : Interface web

Scikit-learn : Modèle SVM pour classification

Mistral AI : Analyse et explication des résultats

NLTK : Prétraitement du texte

Python : Backend

### Fonctionnalités :
Analyse de texte pour détecter fake/real news(par modèle)

Score de confiance en pourcentage

Explications générées par IA

Conseils de vérification

Interface conversationnelle
